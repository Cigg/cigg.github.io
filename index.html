Indirect Lighting Using Voxel Cone Tracing

C++, OpenGL, GLSL

Real time global illumination based on voxel cone tracing. The triangle mesh of the scene is voxelized before it is sampled to get indirect lighting in the scene.

Project on GitHub

Monte Carlo Ray Tracer

C++

A global illumination renderer that uses Monte Carlo integration to numerically solve the rendering equation for global illumination. Supports diffuse objects with different BRDFs, specular objects and refractive objects. Project in the course Advanced Global Illumination and Rendering (TNCG15), at Linköping University.

Smoothed Particle Hydrodynamics

Simulation, C++, OpenGL

Project in the Modeling and Simulation course (TNG022) at Linköping University. Smoothed Particle Hydrodynamics is a fluid simulation technique. The fluid is made up of a large number of particles. How they behave is described by the Navier-Stokes equations that have to be solved. Simulated in real-time in an OpenGL implementation.
